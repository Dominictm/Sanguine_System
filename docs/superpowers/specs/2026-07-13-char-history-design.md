# Спека — история изменений карточки персонажа (git)

Дата: 2026-07-13
Фаза 6 дорожной карты `docs/superpowers/plans/2026-07-13-development-roadmap.md`.

## Задача

Вкладка «История» в модалке персонажа: git-история файла карточки с
просмотром диффов — страховка от неудачной AI-перезаписи.

## Сервер

`web/routes/characters.js`:

- `GET /api/characters/:slug/history` → `{ available, commits: [{hash, date, subject}] }`
  — `git log --follow --format=%h%x09%cs%x09%s -- <файл карточки>` через
  `execFile('git', [...])` (массив аргументов, не shell-строка); cwd — корень
  репо. Ограничение `-n 50`.
- `GET /api/characters/:slug/history/:hash` → `{ diff }` —
  `git show --format=%h%x09%cs%x09%s <hash> -- <файл>`; `:hash` валидируется
  `/^[0-9a-f]{7,40}$/i` (400 иначе).
- Slug валидируется существующим механизмом роутера (резолв карточки → 404).
- **Graceful degradation:** git недоступен / файл вне репо (релизная версия:
  `cities/` пользователя не отслеживается) → `{ available: false, commits: [] }`,
  HTTP 200, не 500. Никаких мутирующих git-команд.

## Фронтенд

`web/public/scripts/char-detail.js`: вкладка «История» в модалке персонажа —
лениво по клику (как «Лист»); `available: false` или пустой список →
«История недоступна (файл вне git)» / «Изменений не зафиксировано»; список
коммитов (дата — тема), клик по коммиту раскрывает дифф моноширинным блоком,
строки `+`/`-` подкрашены классами на существующих токенах.

## Тесты / приёмка

- Integration: history существующего персонажа Парижа → `available: true`,
  commits непусты, у коммита есть hash/date/subject; несуществующий slug →
  404; невалидный hash → 400; diff валидного коммита содержит `diff --git`.
- CDP: вкладка отображает список, клик раскрывает дифф, консоль чистая.
