## Найдено и исправлено по пути

- **Краш в пикере связей** (`archive.js`, обнаружено CDP-прогоном): `x.text.toLowerCase()`
  падал, если у персонажа/локации не заполнено имя (`c.name`/`l.name`
  undefined). Исправлено фильтром `.filter(c => c.name)` перед построением
  списка совпадений.
- Порядок обработчиков в делегированном click-листенере `#chronicle-content`:
  новые кнопки форм используют класс `chron-toggle` для единого стиля, из-за
  чего старая общая ветка `.chron-toggle` (переключение «Подробнее») могла бы
  перехватывать их раньше специфичных обработчиков — специфичные ветки
  (`tl-*`/`ws-*`) размещены выше по коду.
