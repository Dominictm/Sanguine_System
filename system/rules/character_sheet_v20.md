> 📌 Протокол создания и хранения игромеханических листов персонажей по правилам V20.
> ✅ Сверено со структурой `cities/<город>/`. `<slug>` — ASCII-слаг персонажа (= имя папки).
> 🗂️ Поля, названия характеристик и разделы — строго по бланку **STV2098** (интерактивный, ред. 1.4, А. Фиглин). Русские термины ниже — канон проекта; англ. в скобках даётся для однозначности.

## 📂 Хранение

Лист хранится в папке-слаге персонажа, рядом с карточкой:

```
cities/<город>/characters/<линейка>/<slug>/
├── <slug>.md          ← нарративная карточка
└── <slug>-sheet.md    ← игромеханический лист (этот протокол)
```

В карточке персонажа добавить строку после заголовка:
```
> 🎲 [Лист персонажа](<slug>-sheet.md)
```

## 🔁 Протокол создания листа

### Шаг 1 — Шапка (концепция)

Поля шапки бланка (слева — личные данные, справа — архетипы и клан):

| Поле | Значение |
|---|---|
| **Игрок** | имя игрока (для НПС — «—» или «Рассказчик») |
| **Имя** | имя персонажа |
| **Поколение** | обычно 8–13 для стартовых ПК |
| **Сир** | имя сира |
| **Натура (Nature)** | архетип внутреннего «я» |
| **Маска (Demeanor)** | публичная маска — как персонаж себя подаёт |
| **Амплуа (Concept)** | концепция: кем был / кто он по роли |
| **Клан (Clan)** | клан |

### Шаг 2 — Характеристики (все начинают с 1)

Расставить приоритет: **Физические / Социальные / Ментальные**

| Приоритет | Дополнительных точек |
|-----------|---------------------|
| Первичный | 7 |
| Вторичный | 5 |
| Третичный | 3 |

### Шаг 3 — Способности (все начинают с 0, max 3 при создании)

Расставить приоритет: **Таланты / Навыки / Знания**

| Приоритет | Точек |
|-----------|-------|
| Первичный | 13 |
| Вторичный | 9 |
| Третичный | 5 |

### Шаг 4 — Преимущества
- **Дисциплины:** 3 точки (только клановые при создании)
- **Факты биографии (Backgrounds):** 5 точек
- **Добродетели:** 7 точек (Совесть/Решимость, Самоконтроль/Инстинкты, Смелость; каждая начинает с 1)

### Шаг 5 — Производные характеристики (завершающие штрихи)
- **Человечность/Путь** = Совесть + Самоконтроль. У персонажа либо Человечность, либо Путь — **никогда оба сразу**. Человечность — значение «по умолчанию».
- **Путь Просвещения** (если выбран вместо Человечности): альтернативные добродетели (**Убеждённость** и/или **Инстинкты**) начинают с **0**; Сила воли должна быть **не меньше 5**; значение Пути — **не выше 5**. В поле «столп» вписывается название Пути.
- **Сила воли (Воля)** = значение Смелости (постоянная); временный пул = столько же квадратов.
- **Запас крови** — по поколению (таблица ниже); «предел траты в ход» — оттуда же.

### Шаг 6 — Свободные очки: 15

Сноска бланка: `характеристики 7/5/3 • способности 13/9/5 • дисциплины 3 • факты биографии 5 • добродетели 7 • свободные пункты 15 (5/2/7/1/2)`

| Характеристика | Стоимость за точку |
|---|---|
| Характеристика (атрибут) | 5 |
| Способность | 2 |
| Дисциплина | 7 |
| Факт биографии | 1 |
| Добродетель | 2 |
| Человечность/Путь | 1 |
| Воля | 1 |

### Распределение очков по типу вампира (канон: «Создание каинита», V20)

Стандартное распределение для **неонатов/птенцов** (старшим возраст может менять — дополнение «Возраст»). Ни одну Черту нельзя поднимать выше 5; выше 5 — только при поколении 7-го и ниже.

| Тип персонажа | Характеристики | Способности | Дисциплины | Факты биографии | Добродетели | Свободные очки |
|---|---|---|---|---|---|---|
| Вампир Камарильи (кроме шабаша/анархов) | 7/5/3 | 13/9/5 | 3 | 5 | 7 | 15 |
| Вампир-анарх | 6/5/3 | 12/8/5 | 4 | 6 | 7 (5 для бывших шабашитов) | 18 |
| Шабашит | 7/5/3 | 13/9/5 | 4 | 0 | 5 | 15 |

> По умолчанию для ПК/НПС проекта — строка **Камарильи** (7/5/3 · 13/9/5 · 3 · 5 · 7 · 15). Анарх/шабаш — только если это прямо следует из карточки.

### Шаг 7 — Специфика концепции (обязательно)

Лист обязан **отражать концепцию и роль персонажа** — характеристики не равномерные, а смещены под то, кем персонаж является и что умеет. Сначала определи 2–4 «профильных» способности/характеристики из амплуа, роли в модуле и биографии, и подними их выше среднего; всё непрофильное держи низким.

Примеры соответствия (ориентир, не догма; названия — по бланку):

| Амплуа / роль | Профильные характеристики (высокие) |
|---|---|
| Водитель / гонщик | **Вождение**, Ловкость, Смекалка |
| Стрелок / киллер | **Стрельба**, Восприятие, Самоконтроль |
| Уличный боец / вышибала | **Драка**, Сила, Выносливость, Запугивание |
| Охотник на вампиров | Стрельба/Фехтование, Оккультизм, Расследование, Бдительность |
| Светский манипулятор / политик | Манипуляция, Обаяние, Этикет, Хитрость, Политика |
| Хакер / техник | Информатика, Электроника, Интеллект, Естественные науки |
| Врач / медик | Медицина, Интеллект, Эмпатия |
| Вор / взломщик | Воровство, Скрытность, Ловкость |
| Учёный / оккультист | Оккультизм, Гуманитарные науки, Расследование |
| Информатор / связной | Уличное чутьё, Хитрость, Контакты (факт биографии) |

Правила:
- Профильная способность персонажа-эксперта — обычно **3–5** (для матёрого НПС), любителя — 2–3.
- Не давай высокий балл тому, что не следует из концепции (стрелку не нужны Гуманитарные науки 4).
- Дисциплины — по клану и роли; у охотника-смертного дисциплин нет, но есть профильные навыки.
- Поколение, клановая слабость и клановые дисциплины — строго по справочным таблицам ниже.
