# 🩸 Sanguine System

Локальный менеджер хроник для **Vampire: The Masquerade V20** (и смежных линеек WoD).
Инструмент Рассказчика: персонажи, хроники, модули, локации, политика — всё в одном месте.
ИИ-генерация прозы, реплик НПС, сценариев и листов V20 через Claude / Gemini / GPT / OpenRouter.

> 📖 **Полное руководство** — [docs/guide.md](docs/guide.md)

---

## Быстрый старт

```
Двойной клик → start.bat
```

Браузер откроется на `http://localhost:3000`. Node.js и Git установятся автоматически при первом запуске.

**Системные требования:** Windows 10+ x64, современный браузер.

## Возможности

| Раздел | Что делает |
|--------|------------|
| **Панель** | Статистика города, открытые нити, активность |
| **Хронология** | Таймлайн событий с фильтрами по хроникам |
| **Персонажи** | Карточки, биографии, дневники, листы V20, реплики |
| **Связи** | Интерактивный граф отношений (D3.js) |
| **Фракции** | Политическая карта: альянсы, вражда, нейтралитет |
| **Хроники** | Управление хрониками, рекап для игроков |
| **Модули** | Сессии и сценарии: фазы A → B → C |
| **Нити** | Открытые сюжетные нити по всем хроникам |
| **Слухи** | d20-таблицы слухов Элизиума и Грёз |
| **Локации** | Атмосфера, VtM-контекст, крючки |
| **Библиотека** | Справочник дисциплин V20 и психических способностей Numina |
| **Инструменты** | Создание контента, лог сессии, валидация, настройки ИИ |
| **Поиск** | Полнотекстовый поиск по городу |

## ИИ-провайдеры

Каждая функция генерации привязывается к провайдеру независимо (**Инструменты → Модели AI → ⚡ Назначение провайдеров**). Достаточно одного провайдера.

| Провайдер | Лучше для | Что нужно |
|-----------|-----------|-----------|
| **Claude** (Anthropic) | Художественная проза, нарратив | Вход через Claude Code или `ANTHROPIC_API_KEY` |
| **Gemini** (Google) | Проза, длинные сценарии | `GEMINI_API_KEY` + TgWsProxy — [инструкция](docs/gemini.md) |
| **OpenRouter** | Vision-анализ арта, резервный | `OPENROUTER_API_KEY` (есть бесплатные модели) |
| **GPT** (OpenAI) | Универсальный | `OPENAI_API_KEY` |

Ключи хранятся в `web/.env` (не в git). Настраиваются прямо в UI.

**При лимите (429)** генерация автоматически переключается на резервного провайдера и помечает, что ответ пришёл от резерва.

## Структура

```
cities/<город>/          ← данные города (персонажи, хроники, локации, архив)
system/                  ← правила, справочники, схемы (общие для всех городов)
tools/                   ← CLI-утилиты
web/                     ← сервер Express + SPA-фронтенд
docs/                    ← документация
start.bat                ← точка входа (Windows)
```
