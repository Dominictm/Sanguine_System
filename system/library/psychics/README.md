# Библиотека психических способностей (Numina)

Канонический источник справочника психических способностей: **по одному MD-файлу
на способность** (категорию/«нумен»). Сервер (`web/lib/psychics.js` →
`parsePsychicMd`) парсит эти файлы, отдаёт по `GET /api/library/psychics`, веб
рендерит (страница «Библиотека» → вкладка «Психические способности»). Уровень
силы в вебе показывается красно-золотыми точками (1 уровень = 1 точка, до 5 —
у психических способностей шкала канонически 1–5, без расширений уровня 6,
которые есть у дисциплин).

Формат файла полностью аналогичен `system/library/disciplines/README.md` —
только метаполе принадлежности другое (психики — не кланы, а свободный дар
смертных), и нет конструкции «Путь» (у психических способностей в каноне нет
путей — только плоская шкала 1–5).

## Формат файла

```markdown
# 🧠 Биоконтроль (Biocontrol)

- **Категория:** контроль над телом
- **Бросок:** Выносливость + Медитация
- **Источник:** https://wod.fandom.com/ru/wiki/Биоконтроль

> (опц.) Примечание способности — строки-цитаты до первого «## ».

---

## Уровень 1 — Самоконтроль (Self-Control)

**Литературное описание.** <флавор-текст>

**Система.** <механика: пул, сложность, эффект>

---
```

Поля шапки:

- **Категория** — короткая смысловая метка способности (аналог «Клан /
  принадлежность» у дисциплин, но без кланов — психические способности не
  привязаны к фракции, ими может обладать любой смертный с даром). Свободный
  текст, 2–4 слова: «контроль над телом», «чтение прошлого через предметы»,
  «движение разума» и т.п.
- **Бросок** — базовая формула броска способности (Атрибут + Способность),
  если на странице-источнике она указана общей для всей способности (не для
  каждого уровня отдельно). Опционально — не у всех способностей есть единая
  формула.
- **Источник** — ссылка на конкретную статью wod.fandom.com (RU), из которой
  адаптирован текст. Текст — добросовестный пересказ своими словами, не
  копипаста.

Парсер `parsePsychicMd` переиспользует ту же логику разбора H2-секций
«Уровень N — …», что и `parseDisciplineMd`, но не ищет вложенные H3-Пути —
у психических способностей плоская шкала 1–5.

## Статус наполнения

Все файлы наполнены полным контентом (литература + система) на основе
[wod.fandom.com/ru/wiki/Психические_способности](https://wod.fandom.com/ru/wiki/Психические_способности)
и страниц отдельных способностей, на которые она ссылается (раздел
«Перечень»). Канонический первоисточник — **Sorcerer Revised Edition** (для
большинства способностей) и **Kinfolk: Unsung Heroes** (для Эмпатического
исцеления) — указаны как `Источник` в карточках самой вики и продублированы
здесь в примечаниях, где это уместно.

> Примечание: `README.md` сервером пропускается (не способность).
