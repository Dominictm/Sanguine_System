# Библиотека дисциплин V20

Канонический источник справочника дисциплин: **по одному MD-файлу на дисциплину**.
Сервер (`web/lib/disciplines.js` → `parseDisciplineMd`) парсит эти файлы, отдаёт по
`GET /api/library/disciplines`, веб рендерит (страница «Библиотека» и справочник в листе V20).
Уровень силы в вебе показывается красно-золотыми точками (1 уровень = 1 точка, до 10).

## Формат файла

```markdown
# 🐺 Анимализм (Animalism)

- **Клан / принадлежность:** общая (Гангрел, Носферату, Равнос)
- **Источник:** https://wod.su/vampire/disciplines/animalism

> (опц.) Примечание дисциплины — строки-цитаты до первого «## ».

---

## Уровень 1 — Шепоты Зверя (Feral Whispers)

**Литературное описание.** <флавор-текст>

**Система.** <механика: пул, сложность, эффект>

---
```

**Дисциплины с Путями** (Некромантия, Тауматургия — нет общей шкалы 1–5): вместо плоских
`## Уровень N` идут группы-Пути, внутри — силы H3:

```markdown
## Путь Склепа

> (опц.) примечание пути

### Уровень 1 — Прозрение (Insight)

**Литературное описание.** …

**Система.** …
```

Парсер различает: H2 вида `Уровень N — …` → плоская сила; любой другой H2 с вложенными
H3-силами `Уровень N — …` → группа-Путь (имя пути может и не содержать слова «Путь»,
напр. «Привлечение Огней», «Руки Разрушения»).

## Статус наполнения

| | |
|---|---|
| Полный контент (литература + система) | `animalism.md` |
| Скелеты (система перенесена из справочника V20, **«Литературное описание» — ⏳ TODO с wod.su**) | остальные 16 |

Скелеты сгенерированы из прежнего `RULES_DISCIPLINES`; имена сил, уровни, кланы и краткая
«Система» — корректны. Поле «Литературное описание» предстоит наполнить добросовестной
адаптацией с [wod.su/vampire/disciplines](https://wod.su/vampire/disciplines) — пометка
источника в каждом файле. Уровни 1–5 — корбук V20, уровни 6 — расширенный «доп. канон»
(кланбуки), не обязателен для НПС.

> Примечание: `README.md` сервером пропускается (не дисциплина).
