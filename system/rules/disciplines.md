## ⚠️ Канонический статус по уровням

- **Уровни 1–5** (●○○○○ … ●●●●●) — базовая механика V20-корбука. Для трёх «безымянных» дисциплин (Стремительность, Стойкость¹, Могущество) на этих уровнях нет именных сил — эффект числовой (бонус-кубики/действия). ¹Стойкость на wod.su даёт именные альтернативные силы 1–4 из дополнительных источников (см. таблицу ниже) — отыгрывать можно либо как чистый бонус к Стойкости, либо как один из этих эффектов.
- **Уровни 6+** — расширенный канон из дополнительных книг (кланбуки, *Guide to the Camarilla*, антологии), не входит в стандартный набор уровней 1–5 V20-корбука. Используй с пометкой «доп. канон» и проверяй применимость к конкретной хронике; не считается обязательным для НПС.
- Некромантия и Тауматургия **не имеют собственной шкалы уровней** — механика реализуется только через Пути (Paths), каждый со своей иерархией 1–5.

## 📚 Применение

- При создании НПС-вампира (протокол [`npcs_city.md`](npcs_city.md)) — клановые дисциплины бери из таблицы [`reference_wod.md`](reference_wod.md), конкретные силы по уровню — из этого файла.
- В листе персонажа (раздел «Дисциплины» в [`character_sheet_v20.md`](character_sheet_v20.md)) — записывай дисциплину и точки (●●●○○), конкретную силу указывай в описании/механике сцены при использовании.
- Силы уровня 6+ — отыгрывай только для значимых старейшин/сюжетных фигур, с пометкой «доп. канон», и проверяй уместность для конкретной хроники.
- 📖 Источник: [wod.su/vampire/disciplines](https://wod.su/vampire/disciplines), сверено вручную (Fortitude, Potence — уточнены повторным запросом из-за неоднозначной разметки уровней на странице). Для Анимализма и Помешательства точная градация уровней выше 6 на странице не подтвердилась однозначно — все силы сгруппированы под уровнем 6 с пометкой в тексте.
