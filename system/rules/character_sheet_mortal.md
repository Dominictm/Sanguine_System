# ⚙️ СИСТЕМНОЕ ПРАВИЛО: Лист персонажа — Смертный (cWoD / V20)

> 📌 Протокол создания игромеханического листа **смертного** персонажа (линейка `mortals`).
> 🗂️ Поля, названия характеристик и базовые разделы — по бланку **STV2098** (общая основа cWoD). Термины ниже — канон проекта; англ. в скобках для однозначности.
> ⚠️ Смертный — **не** сородич: у него **нет** дисциплин, запаса крови, поколения, Пути. Только Человечность.

### Шаг 4 — Преимущества
- **Дисциплины:** 0 (смертный не имеет вампирских сил)
- **Факты биографии (Backgrounds):** 5 точек
- **Добродетели:** 7 точек (Совесть/Решимость, Самоконтроль/Инстинкты, Смелость; каждая с 1)

### Шаг 5 — Производные характеристики
- **Человечность** = Совесть + Самоконтроль
- **Сила воли (Воля)** = значение Смелости; временный пул = столько же
- Запаса крови **нет**

### Шаг 6 — Свободные очки: 21

| Характеристика | Стоимость за точку |
|---|---|
| Характеристика (атрибут) | 5 |
| Способность | 2 |
| Факт биографии | 1 |
| Добродетель | 2 |
| Человечность | 1 |
| Воля | 1 |

### Шаг 7 — Специфика концепции (обязательно)

Лист обязан отражать роль персонажа: 2–4 профильные способности выше среднего, остальное низко. Криминальный авторитет — Запугивание, Лидерство, Хитрость, Уличное чутьё; врач — Медицина, Интеллект; силовик — Стрельба, Драка, Бдительность. Эксперт-смертный обычно 3–4 в профиле (5 — выдающийся мастер).

### Шаг 8 — Сверхъестественное (опционально)

Обычный смертный сил не имеет. Если персонаж — **Охотник (Имбьюэд)**, экстрасенс или практик худо-магии, добавь раздел «Нумина / Грани» (см. шаблон); точные правила — по источнику (*Hunter: The Reckoning*, *Sorcerer*), помечай «требует сверки».

#### Психические способности (экстрасенс)

Если персонаж — **экстрасенс/психик** (не Имбьюэд и не маг-практик), его силы берутся из справочника `system/library/psychics/` (Психометрия, Телепатия, Психокинез, Биоконтроль, Прорицание/предвидение и т.д. — список см. `system/library/psychics/README.md`). В раздел «Нумина / Грани» листа заносится 1–3 способности с точками.

**Точки на психические способности при создании:** свободного канонического пула для «обычного смертного-психика» в материалах STV2098/V20 нет — психические силы в каноне обычно приходят через Достоинство (Merit) «Психический» («Psychic», как правило 1–3 точки Достоинства) или через статус Охотника-Имбьюэд (отдельная линейка, не этот протокол). Для проекта принят следующий дефолт:

- ⚠️ **Нужна сверка канона.** Пока не подтверждено source-пейджем V20/STV2098 — используется рабочее правило проекта: **3 точки**, распределяемые свободно по 1–3 способностям (зеркало пула Дисциплин у сородичей, Шаг 4 листа вампира), но **списываются из Достоинств/Свободных очков персонажа** (Шаг 4/6 выше), а не выдаются сверх общего бюджета — т.е. психик не получает дополнительных точек по сравнению с обычным смертным, он просто тратит часть уже имеющихся свободных/достоинство-очков на способности вместо Достоинств или Способностей.
- Максимум одной способности при создании — **3** (как и обычные способности смертного, Шаг 3); до 5 — только через Достоинство «Психический» более высокого ранга или развитие опытом (XP) после создания.
- Если у персонажа есть подходящее Достоинство (Merits & Flaws, см. шаблон ниже) — явно укажи его и можно поднять одну способность до уровня, который покрывает ранг Достоинства.

## ⚠️ Ограничения и примечания

- При создании: характеристики max 5, способности max 3 (без свободных очков)
- У смертного нет дисциплин, запаса крови, поколения, клановой слабости и Пути
- Большинство смертных проекта — **эпизодические НПС**: достаточно профильных способностей и пары фактов биографии; полный лист — для значимых фигур
- Если персонаж становится гулём — добавляется Потенция (1 дисциплина от регента) и запас крови; см. отдельные правила гулей
- Психические способности (раздел «Нумина / Грани») — **не у всех смертных**; точечный пул для них (Шаг 8) — ⚠️ не подтверждённое канонической страницей рабочее правило проекта, требует сверки источника при первой серьёзной ревизии экстрасенса
