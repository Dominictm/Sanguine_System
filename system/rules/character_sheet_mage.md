> ⚠️ У мага **нет** дисциплин и запаса крови. Вместо Человечности — Парадокс; магические способности — Сферы (Spheres), сила магии — Арете.

## 🔁 Протокол создания листа

### Шаг 1 — Шапка (концепция)

| Поле | Значение |
|---|---|
| **Игрок** | имя игрока (для НПС — «—» / «Рассказчик») |
| **Имя** | имя персонажа |
| **Натура (Nature)** | архетип внутреннего «я» |
| **Маска (Demeanor)** | публичная маска |
| **Амплуа (Concept)** | концепция: кто персонаж по роли |
| **Традиция / Техно / Нефандус** | Орден Гермеса / Культ Экстаза / Сфера Разума / Вербена / Эутан. Инж. / и др. |
| **Аватар** | имя/образ духа-наставника |
| **Парадигма** | как маг понимает магию (наука / мистика / вера / …) |

### Шаг 2 — Характеристики (все начинают с 1)

Приоритет: **Физические / Социальные / Ментальные**

| Приоритет | Дополнительных точек |
|-----------|---------------------|
| Первичный | 7 |
| Вторичный | 5 |
| Третичный | 3 |

### Шаг 3 — Способности (все начинают с 0, max 3 при создании)

Приоритет: **Таланты / Навыки / Знания**

| Приоритет | Точек |
|-----------|-------|
| Первичный | 13 |
| Вторичный | 9 |
| Третичный | 5 |

### Шаг 4 — Преимущества

- **Сферы (Spheres):** 6 точек (распредели по Сферам; ни одна выше 3 при создании без Арете ≥3)
  - Доступные сферы: Разум (Mind), Сила (Forces), Материя (Matter), Жизнь (Life), Пространство (Correspondence), Дух (Spirit), Время (Time), Первопричина (Prime), Энтропия (Entropy)
- **Факты биографии (Backgrounds):** 7 точек
  - Доступны: Аватар, Библиотека, Одержание, Менторство, Ресурсы, Союзники, Орден, Репутация
- **Добродетели:** как у смертного (Совесть, Самоконтроль, Смелость; по 7 точек)

### Шаг 5 — Специальные характеристики мага

| Характеристика | Начальное значение | Описание |
|---|---|---|
| **Арете (Arete)** | 1 | Мастерство магии; максимум 10. Стоит 4 фриби/ХО за точку. |
| **Квинтэссенция (Quintessence)** | = значение фона «Аватар» | Пул магической энергии; восполняется у Узлов |
| **Парадокс (Paradox)** | 0 | Нарастает при грубой магии на виду; разряжается событием/кризисом |
| **Воля (Willpower)** | 5 | Временный пул = столько же |

### Шаг 6 — Производные характеристики

- **Здоровье:** 7 уровней (Царапина → Искалечен → Недееспособен)
- **Инициатива** = Ловкость + Самоконтроль
- **Скорость** = Сила + Ловкость + 5

### Шаг 7 — Свободные очки: 15

| Характеристика | Стоимость за точку |
|---|---|
| Характеристика (атрибут) | 5 |
| Способность | 2 |
| Сфера | 7 |
| Факт биографии | 1 |
| Добродетель | 2 |
| Арете | 4 |
| Квинтэссенция | 1 |
| Воля | 1 |

### Шаг 8 — Специфика концепции (обязательно)

Лист должен отражать Традицию и специализацию: Орден Гермеса — Разум/Сила, высокая Интеллект; Культ Экстаза — Время/Жизнь, Восприятие; Сфера Разума — Материя/Сила, Технология. Арете обычно 1–3 для начинающего, 4–5 для зрелого мага. Если маг — НПС в VtM-хронике, акцент на 2–3 Сферах, остальное минимально; Парадокс 0–3 (нарастает по ходу хроники).
