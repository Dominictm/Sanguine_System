# Настройка Google Gemini AI

Gemini используется для генерации прозы, реплик, сценариев и финалов.
Модели **gemini-2.5-flash** (быстрая) и **gemini-2.5-pro** (качество).

---

## Почему нужен прокси

`generativelanguage.googleapis.com` — Google API — в ряде регионов недоступен напрямую.
Для работы Gemini через сервер Sanguine нужен локальный HTTP-прокси.

---

## Шаг 1 — TgWsProxy (локальный прокси-туннель)

В корне проекта находится `TgWsProxy_windows.exe` — инструмент,
поднимающий локальный HTTP-прокси на `127.0.0.1:12334`.

> Если файл отсутствует — скачай актуальную версию со страницы релизов:
> <https://github.com/Flowseal/tg-ws-proxy/releases>
> Файл `TgWsProxy_windows.exe` положи в корень проекта (рядом с `start.bat`).

### Первый запуск

1. Запусти `TgWsProxy_windows.exe` вручную один раз — откроется окно с инструкцией.
2. Приложение свернётся в системный трей.
3. Убедись, что иконка появилась в трее и прокси поднялся.

### Последующие запуски

`start.bat` запускает TgWsProxy автоматически при каждом старте сервера
(если он ещё не запущен). Закрывать TgWsProxy вручную не нужно.

---

## Шаг 2 — Ключ Gemini API

Получи API-ключ на [Google AI Studio](https://aistudio.google.com/apikey):

1. Войди в аккаунт Google.
2. Нажми **«Create API key»**.
3. Скопируй ключ (начинается с `AIza...`).

---

## Шаг 3 — Настройка `web/.env`

Открой файл `web/.env` и добавь (или раскомментируй) строки:

```
GEMINI_API_KEY=AIza...сюда_вставь_свой_ключ...
GEMINI_MODEL=gemini-2.5-flash
HTTPS_PROXY=http://127.0.0.1:12334
```

`GEMINI_MODEL` — опционально, дефолт `gemini-2.5-flash`.
Для более качественной прозы смени на `gemini-2.5-pro` (медленнее).

---

## Шаг 4 — Перезапуск сервера

После правки `.env` перезапусти сервер:
- через браузер: **Инструменты → Модели AI → Перезапустить сервер**
- или закрой и снова открой `start.bat`

При старте в консоли должна появиться строка:
```
[proxy] fetch routed via http://127.0.0.1:12334
```

---

## Шаг 5 — Назначение провайдера

В браузере: **Инструменты → Модели AI → ⚡ Назначение провайдеров**

Выбери **Gemini** для нужных функций:
- **Генерация прозы** — дневники, финалы сессий
- **Генерация реплик НПС**
- **Генерация сценариев модулей**

---

## Проверка

Открой любую карточку персонажа → **Реплики НПС** → выбери провайдер **Gemini** → введи ситуацию → нажми «Сгенерировать».

Если появляется ошибка:

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `GEMINI_API_KEY не задан` | Ключ не добавлен в `.env` | Добавь `GEMINI_API_KEY=...` в `web/.env` |
| `UND_ERR_CONNECT_TIMEOUT` | Прокси не запущен | Убедись что TgWsProxy запущен (иконка в трее), перезапусти сервер |
| `Нет доступа к Google Gemini API` | Порт прокси другой | Проверь в Настройках → Сеть что прокси на `127.0.0.1:12334`; если другой — обнови `HTTPS_PROXY` в `.env` |
| `403 Forbidden` | Неверный ключ | Проверь ключ в [AI Studio](https://aistudio.google.com/apikey) |
| `Gemini вернул пустой ответ` | Запрос заблокирован safety-фильтрами | Не должно происходить (фильтры отключены); попробуй другую формулировку |
