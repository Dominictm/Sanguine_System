# Спека — дашборд «висящих нитей»

Дата: 2026-07-13
Фаза 2 дорожной карты `docs/superpowers/plans/2026-07-13-development-roadmap.md`.

## Задача

Нити из `open_threads.md`, которые давно не двигались, видны на дашборде —
забытые сюжеты не теряются.

## Ключевое решение: давность в игровом времени

Колонка «Источник» нитей содержит игровую дату упоминания
(`«Кошки и мышки», ноябрь 2010`, `январь 2011 ⟨котерия ДНП⟩`). Давность
считается **в игровых месяцах** относительно самой свежей нити города:

- из `source` берётся **последнее** вхождение пары «месяц год»
  (русские месяцы в любом падеже по основе: янв/фев/мар/…; год — 4 цифры);
- точка отсчёта — максимальная такая дата среди всех нитей города;
- `staleMonths = (refYear*12+refMonth) − (year*12+month)`;
- нити без распознаваемой даты → `sourceDate: null`, `staleMonths: null`
  (не ранжируются, в топ не попадают).

Отвергнутая альтернатива — `git log -1` по файлу нитей: дата общая на весь
файл и сбрасывается любой правкой любой нити; в релизной версии
пользователя `cities/` вообще не под git.

## Сервер

- `web/lib/parsers/threads.js`: новая чистая функция
  `threadSourceDate(source)` → `{year, month} | null`. Экспортируется.
- `web/routes/threads.js` (`GET /api/threads`): каждому элементу добавляются
  поля `sourceDate` и `staleMonths` (аддитивно, существующие потребители не
  ломаются). Считается внутри `readThreadsStructured` или в роуте.
- Новых эндпойнтов нет.

## Дашборд

`renderDashboard` (`web/public/scripts/scripts.js`): под `.substats` — блок
«🧵 Висящие нити»:

- топ-5 незакрытых нитей (`status` ∈ {active, background}) с
  `staleMonths ≥ 1`, сортировка по `staleMonths` убыв.;
- строка: приоритет/статус-точка, название, «N мес. без движения»
  (правильная русская плюрализация: месяц/месяца/месяцев), источник;
- клик по строке → страница «Нити» (`navigate('threads')`);
- данные — параллельный `fetch('/api/threads')` в `loadDashboard`; при
  ошибке или пустом топе блок не показывается (дашборд не ломается);
- CSS — существующие токены; touch-цели строк ≥44px на `pointer: coarse`
  (правило CLAUDE.md; заодно первая ласточка дорожки T.1).

## Тесты / приёмка

- Unit: `threadSourceDate` — «…, ноябрь 2010», «январь 2011 ⟨…⟩», два
  вхождения дат (берётся последнее), без даты → null, только год без
  месяца → null.
- Integration: `GET /api/threads?city=paris` — элементы имеют
  `staleMonths`, у самой свежей нити 0, у более старой > 0.
- CDP: блок отображается на дашборде Парижа, клик ведёт на «Нити»,
  консоль чистая.
