## 🔁 Протокол создания листа

### Шаг 1 — Концепция и фейская природа

| Поле | Значение |
|---|---|
| **Игрок** | имя игрока (для НПС — «—») |
| **Имя** | имя персонажа |
| **Кит (Kith)** | вид фейри (даёт Преимущество + обязательную Слабость, см. справочник): Богган, Гилли Ду, Красная Шапка, Нокер, **Пак (Pooka)**, Сатир, Слуа, Тролль, Ши (Сидхе), Эшу |
| **Обличье (Seeming)** | возрастная стадия души: **Дитя (Childling) / Юноша·Вильдер (Wilder) / Старец·Гранд (Grump)** |
| **Двор / Легаси** | **Благой (Seelie)** или **Неблагой (Unseelie)** Двор (легаси) |
| **Дом (House)** | только для Сидхе |
| **Натура (Nature)** | архетип внутреннего «я» |
| **Маска (Demeanor)** | публичная маска |
| **Амплуа (Concept)** | концепция смертной жизни |
| **Фригольд** | дом-убежище фейской силы (если есть) |

### Шаг 2 — Характеристики (все начинают с 1)

Приоритет: **Физические / Социальные / Ментальные** — **7 / 5 / 3**.

### Шаг 3 — Способности (все начинают с 0, max 3 при создании)

Приоритет: **Таланты / Навыки / Знания** — **13 / 9 / 5**.

### Шаг 4 — Преимущества
- **Сферы / Королевства (Realms):** 5 точек — над кем/чем фея властна (Люди·Актёр, Феи, Природа, Предметы, Места·Сцена, Время)
- **Искусства (Arts):** 3 точки — как творит магию; 6 базовых корбука: Обман/Chicanery, Фокусы/Legerdemain, Изначальность/Primal, Предсказание/Soothsay, Правление/Sovereign, Странствия/Wayfare
- **Факты биографии (Backgrounds):** 5 точек

### Шаг 5 — Производные и фейские характеристики
- **Glamour (Грёза)** — топливо фейской магии; стартовое значение по Обличью (таблица ниже)
- **Banality (Банальность)** — оцепенение смертного мира; стартовое значение по Обличью
- **Сила воли (Воля)** — стартовое значение по Обличью
- **Человечность** — для подменыша обычно не отслеживается (вместо неё пара Glamour/Banality); вести по решению Рассказчика

**Стартовые значения по Обличью** (по умолчанию рода/двора; max любой Черты Характера — 10):

### Преимущества и Слабости родов (Birthright / Frailty)

Каждый род даёт **Преимущество** и **обязательную Слабость** (аналог клановой слабости). Гилли Ду — редкий род вне основной девятки (по решению Рассказчика).

| Род | Преимущество (Birthright) | Слабость (Frailty) |
|---|---|---|
| Богган | Ремесло (работает ×3 быстрее без надзора, не проваливает Ремесло); чутьё на отношения в группе | **Зов долга:** не может пройти мимо чужой беды — Воля (сл. 8), иначе бросается помогать |
| Эшу | Духовные пути (не теряется в дороге, всегда находит маршрут); мастер-рассказчик (не проваливает Исполнение/Эмпатию) | **Безрассудство:** не может отказаться от вызова, если есть шанс выжить |
| Нокер | Выковывание химер и починка механизмов (Интеллект+Ремесло) | **Недостатки:** всё созданное имеет неустранимый мелкий дефект |
| Пак | Преображение в животное за 1 ход (1 Гламур); чтение душ (Восприятие+Эмпатия) | **Ложь:** не может сказать прямую ложь без броска Воли (сл. 8) |
| Красная Шапка | Тёмные аппетиты (ест что угодно, укус Сила+2); устрашение (−1 сложность, действует на химер) | **Плохое отношение:** +2 сложности на соц. броски (кроме Запугивания) |
| Сатир | Дар Пана (музыка снижает Банальность на 1–2); физ. мощь (+1 Выносливость, высокая скорость, не проваливает Атлетику) | **Проклятье страсти:** перепады настроения, +2 Воле против желаний |
| Ши (Сидхе) | Благоговение и красота (+2 Привлекательность, не проваливает Этикет); колдовство не выставит дураком | **Проклятье Банальности:** временная Банальность считается удвоенной |
| Слуа | Гибкость (узкие лазы); обострённые чувства (−2 Восприятию, видит иллюзии, не проваливает Бдительность/Скрытность) | **Проклятье молчания:** говорит только шёпотом, +2 соц. сложность |
| Тролль | Сила титана (+здоровье/Сила у Юношей/Старцев); упорство (+2 кубика Воле) | **Оковы долга:** нарушение клятвы ослабляет, предательство — Воля (сл. 8) против ярости |

### Искусства (Arts)
**6 базовых Искусств корбука:** Обман (Chicanery), Фокусы (Legerdemain), Изначальность (Primal), Предсказание (Soothsay), Правление (Sovereign), Странствия (Wayfare).
Из дополнений (с одобрения Рассказчика): Именование (Naming), Ремесло Историй (Tale Craft), Инфузия (Infusion), Метаморфозис (Metamorphosis); Inanimae — Пламя (Pyretics), Криос (Kryos), Ремесло Бурь (Skycraft), Афродизия (Aphrodesia).

### Сферы / Королевства (Realms) — 6 корбука
Люди·Актёр (Actor/Humans), Феи (Fae), Природа (Nature), Предметы (Prop/Objects), Места·Сцена (Scene/Locations), Время (Time).

### Дома Ши (Houses) — для кита Ши/Сидхе
Балор (Balor), Бомайн (Beaumayn), Варич (Varich), Гвидион (Gwydion), Дайреанн (Daireann), Дугал (Dougal), Лиам (Liam), Лианнан (Leanhaun), Скатах (Scathach), Фиона (Fiona), Эйзин (Aesin), Эйлиль (Ailil), Эйлунд (Eiluned).

### Дворы (Courts)
- **Благой (Seelie)** — свет, день, лето; старые традиции, законопослушность, красота жизни и возрождения.
- **Неблагой (Unseelie)** — тьма, ночь, зима; слом традиций, перемены, процесс смерти и энтропии.
- **Теневой (Shadow Court)** — тайный двор тьмы внутри Грёзы (по согласованию с Рассказчиком).

## ⚠️ Ограничения и примечания

- При создании: характеристики max 5, способности max 3 (без свободных очков); max любой Черты — 5
- **Банальность нельзя повышать** свободными очками; Черты Характера (Glamour/Banality/Воля) — max 10
- Подменыш живёт в двух мирах: смертная Маска и фейская Личина (Mien); сила черпается из Грёзы (Glamour), угроза — Банальность
- Источник правил создания — *Changeling: The Dreaming* (corebook), глава «Создание персонажа». Числа сверены.
