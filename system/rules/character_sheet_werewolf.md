## 🔁 Протокол создания листа

### Шаг 1 — Шапка (концепция)

| Поле | Значение |
|---|---|
| **Игрок** | имя игрока (для НПС — «—» / «Рассказчик») |
| **Имя** | имя персонажа |
| **Натура (Nature)** | архетип внутреннего «я» |
| **Маска (Demeanor)** | публичная маска |
| **Амплуа (Concept)** | концепция: кто персонаж по роли |
| **Племя (Tribe)** | Серебряные Клыки / Дети Гайи / Феозрачные и др. |
| **Тотем** | духовный тотем персонажа |
| **Каста (Auspice)** | Рагабаш / Тату / Филодокс / Галиард / Коварна |
| **Ранг (Rank)** | 1–5 (Щенок / Ученик / Адепт / Старейшина / Легенда) |

### Шаг 2 — Характеристики (все начинают с 1)

Приоритет: **Физические / Социальные / Ментальные**

| Приоритет | Дополнительных точек |
|-----------|---------------------|
| Первичный | 7 |
| Вторичный | 5 |
| Третичный | 3 |

> Оборотни в форме Криноса получают +4 к Силе, +2 к Выносливости, -2 к Манипуляции — используй значения листа (в форме Хомида) как базу; форма учитывается нарративно.

### Шаг 3 — Способности (все начинают с 0, max 3 при создании)

Приоритет: **Таланты / Навыки / Знания**

| Приоритет | Точек |
|-----------|-------|
| Первичный | 13 |
| Вторичный | 9 |
| Третичный | 5 |

### Шаг 4 — Преимущества

- **Дары (Gifts):** 3 дара — 1 племенной, 1 кастовый, 1 общий (по Рангу 1)
- **Факты биографии (Backgrounds):** 5 точек
  - Доступны: Союзники, Контакты, Дух-страж, Ресурсы, Репутация, Тотем, Родство (Kinfolk)
- **Добродетели:** как у смертного (Совесть, Самоконтроль, Смелость; по 7 точек)

### Шаг 5 — Специальные характеристики оборотня

| Характеристика | Начальное значение | Максимум |
|---|---|---|
| **Ярость (Rage)** | по касте: Рагабаш 1, Тату 4, Филодокс 2, Галиард 3, Коварна 5 | 10 |
| **Гнозис (Gnosis)** | по племени (1–5) | 10 |
| **Воля (Willpower)** | 3 + Смелость | 10 |
| **Репутация (Renown):** Слава / Честь / Мудрость | по 0 | шкала |

> Репутация в очках: укажи начальное значение Славы, Чести и Мудрости по предыстории (обычно 0–3 для молодого оборотня).

### Шаг 6 — Производные характеристики

- **Здоровье:** 7 уровней (Царапина → Искалечен → Недееспособен); в форме Криноса +2 уровня
- **Инициатива** = Ловкость + Самоконтроль
- **Скорость** = Сила + Ловкость + 5 (Хомид), ×2 (Криноc)

### Шаг 7 — Свободные очки: 15

| Характеристика | Стоимость за точку |
|---|---|
| Характеристика (атрибут) | 5 |
| Способность | 2 |
| Дар (за пределами стартового набора) | 7 |
| Факт биографии | 1 |
| Добродетель | 2 |
| Ярость | 1 |
| Гнозис | 2 |
| Воля | 1 |

### Шаг 8 — Специфика концепции (обязательно)

Лист должен отражать касту и племя: Коварна — высокая Ярость, Галиард — Манипуляция и Харизма, Филодокс — Лидерство и Авторитет. Дары подбирай исходя из Ранга и роли в хронике. Если оборотень — НПС в VtM-хронике, сделай акцент на 2–3 ключевых способностях, остальное минимально.
